Notes for pandas

并不是一个pandas的入门博客，仅仅用于记录一些pandas中的一些tips

  1. 对dataframe的单列/多列进行运算。

    对df中的col列进行平方，并将结果保留为new_col列。

    同时进行多列的运算，x对应当前行，通过列名进行索引。

  2. agg, apply和transform

    agg()只做聚合运算

    apply作用对象是dataframe, 与其他两者相比比较灵活

    transform对dataframe的每一个series做transframe操作

  3. df.shift(x)

    数据平移。df.shift(1), 将所有数据向下平移一个单位。对于时间序列数据，df.shift(30), 会使得每一天对应一个月前（三十天前）的数据。

  4. df.rolling(x)

    滑动窗口。对于时序数据，df.rolling(7).mean()可以用于计算过去七天的滑动平均值。
